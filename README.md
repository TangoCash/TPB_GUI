# TP:B GUI 2023
## Code und Design von Benni "PerCenT"


[![TPBGUI.jpg](https://i.postimg.cc/K8vpftfw/TPBGUI.jpg)](https://postimg.cc/CzX4LBWs)


Für optimalen Gebrauch, bitte die Anweisung lesen:

1) Installation
	Entpacke die Datei "https://github.com/TangoCash/TPB_GUI/archive/refs/heads/master.zip" in das Verzeichnis /steamapps/common/Half-Life/. Stelle sicher, dass du alle Dateien überschreibst.
	Ich empfehle jedem, die Mausbeschleunigung zu deaktivieren. Um die Mausbeschleunigung zu deaktivieren, füge -noforcemparms -noforcemaccel deinen Launch Options von Counter-Strike in Steam hinzu.
	Füge dort außerdem noch -freq xxx (xxx = Hertzzahl deines Monitors) hinzu, um mit der höchstmöglichen Bildwiederholfrequenz spielen zu können.

	Im Verzeichnis CFGs/custom befinden sich Config-Dateien die jeder TP:Bler an sich selbst anpassen muss:
	rcon1.cfg = Füge hier das RCON-Passwort des TP:B-Warservers ein (sofern du es kennst)
	nickname1.cfg = Passe hier deinen TP:B-Ingame-Namen an
	nickname2-5.cfg = Hier kannst du Nicht-TP:B-Nicknamen eingeben, zwischen denen du wechseln möchtest

	Im Spiel können noch einige nützliche Einstellungen mit dem Commandmenu gemacht werden. Standardmäßig kannst du ihn im Spiel mit der Taste "F10" aufrufen. 

2) Disclaimer
	Der TP:B GUI ist für ein optimales Spielgefühl angepasst worden.
	Ich empfehle besonders in den Configs nichts umzustellen, sondern Änderungen nur über die Ingame-Einstellungen zu machen.
